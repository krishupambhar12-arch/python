2. Validate model performance using validation data:
python
# Validate Simple Model
y_val_pred_simple = simple_model.predict(X_val[['Reviews']].values)
simple_mse = mean_squared_error(y_val, y_val_pred_simple)  # Using X_val, y_val
 
# Validate Multiple Model
y_val_pred_multi = multi_model.predict(X_val)
multi_mse = mean_squared_error(y_val, y_val_pred_multi)  # Using X_val, y_val
3. Test final model on test data:
python
# Test Simple Linear Regression
y_test_pred_simple = simple_model.predict(X_test[['Reviews']].values)
test_mse_simple = mean_squared_error(y_test, y_test_pred_simple)  # Using X_test, y_test
 
# Test Multiple Linear Regression
y_test_pred_multi = multi_model.predict(X_test)
test_mse_multi = mean_squared_error(y_test, y_test_pred_multi)  # Using X_test, y_test
